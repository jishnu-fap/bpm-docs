Submit a status change request
A project's BPM status moves through defined stages. To change it, open the Status Changes tab and submit a request with a reason. This does not change the status immediately — it goes to an admin for approval.

Review and approve on the Approvals page
Admins open the Project Approvals page from the sidebar. All pending status-change requests appear here. Each card shows the project, the requester, the proposed change, and their reason. The admin can approve or reject with one click.

Project closure
8
Admin
Close a project
Click the ✕ icon on the list row to close a project. This is irreversible — closed projects are hidden by default but can be found by toggling Show closed on the list. Closing does not delete equipment assignments or history.

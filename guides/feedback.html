Feedback Flow

End-to-end journey — from spotting an issue to marking it resolved

1
User
Click the feedback button
On every page in BPM a floating button sits in the bottom-right corner. Any logged-in user can click it at any time — no navigation required.
💬
The button stays visible no matter which page you're on.
2
User Optional
Pick an element on the page
The panel opens. If the feedback is about a specific part of the screen, click "Click to pick an element". A crosshair cursor appears — hover to highlight any element, click to attach it to the report. Press Esc to skip.

Fill in the form & submit

5
Admin
Open the Feedback page
Admin navigates to Feedback in the sidebar. All submissions appear newest-first. Filterable by type and status.

Admin
Expand & investigate
Clicking a row expands it — submitter name & email, exact page URL, and the pinned element (tag, text, CSS selector). "Preview on page" navigates there and highlights it live so the admin sees exactly what was flagged.

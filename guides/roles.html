Create a new role
Click New Role on the Roles list page. Give the role a name, then use the same permission editor to configure what it can access. Start from a blank set or refer to an existing role for guidance — permissions can be individually toggled.

4
Admin
Assign a role to users
Roles are assigned from the Users page — either during user creation or by editing an existing user. One user has exactly one role. To give a user the same access as a colleague, use Clone Permissions from the user kebab menu.
Changes to a role propagate immediately to all users holding that role — no logout required on their side.
5
Admin
Understand permission groupings
Permissions are grouped by domain so it's easy to grant or revoke access to an entire feature area. The key domains and what they control:

Browse contracts with expiry colour coding
Navigate to Contracts. The list is colour-coded to surface risk: rows with a red background have already expired; rows with a yellow background are expiring within 30 days. Use the expiry filter to focus on upcoming renewals.

View contract detail

4
Admin
Monitor expiry and plan renewals
Set the Expiring Within filter to 30, 60, or 90 days to see which contracts need attention. For contracts approaching expiry, create a new quotation referencing the same client and project, get it approved, and convert it to a renewal contract.
There is no automatic renewal — the admin must create a new contract. The expired contract stays on record for reference.
5
Admin
Terminate a contract
From the contract detail page, click Terminate. Enter a termination reason and confirm. The contract status changes to Terminated and the record is preserved for audit purposes. Termination removes the contract from active monitoring.

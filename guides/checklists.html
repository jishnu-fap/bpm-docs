Checklist Templates

How checklist templates are created, structured with sections and items, assigned to projects, duplicated for reuse, and printed for on-site use.

1
Admin / Engineer
Browse checklist templates
Navigate to Checklists in the sidebar. All templates are listed with their name and active/inactive status. Search by name to find existing templates before creating duplicates. Inactive templates are hidden by default — toggle the filter to see them.

Create a new template
Click New Template. Give the template a name and optionally associate it with an equipment type (e.g. Intercom, CCTV). Then add sections — a section is a logical group of checklist items (e.g. "Safety Checks", "Functional Tests", "Sign-off").

Add sections and checklist items
Inside the template, create sections and add items to each. Items are the individual to-do lines that a technician will tick off on-site. Drag to reorder. Each item has a description field — keep these actionable and specific.

4
Admin / Engineer
Duplicate an existing template
Click the Copy icon (⧉) on any template row to create a duplicate. The copy is created as a new active template with "(Copy)" appended to the name. Rename it and adjust the items as needed — useful when equipment types share most of the same checks.
Duplication is the recommended way to build variant templates (e.g. quarterly vs. annual PM) rather than starting from scratch each time.
5
Admin / Engineer
Assign a template to a project
Templates are assigned from the Checklists tab inside a project detail page — not from this template list. Open the project, go to the Checklists tab, click Assign Template, and select from the active template list. One project can have multiple templates assigned.
Once assigned, a copy of the template is attached to the project. Changes to the master template later will not retroactively update assigned project checklists.
6
Engineer (on-site)
Print the checklist for on-site use
From the assigned checklist inside the project, click Print. A print-friendly A4 layout is generated with all sections and items and a checkbox next to each line. The technician ticks items off on paper; the completed sheet is then scanned and attached to the project as evidence.

7
Admin
Deactivate an outdated template
Click the Eye-off icon on a template row to deactivate it. Inactive templates no longer appear in the assignment picker when adding checklists to projects. Existing project assignments are not affected. Reactivate at any time by clicking the Eye icon.
Deactivate rather than delete — this preserves the template for historical reference and keeps assigned project checklists intact.

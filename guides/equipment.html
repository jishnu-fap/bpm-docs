Serial Numbers tab — track individual units
Each physical unit of the equipment gets its own serial number record. Status tracks its condition: Active, Faulty, or Removed. Serials can be linked to a specific project and location.

6
Admin / permitted user
Edit the equipment record
Click the Edit button from the detail page or the pencil icon from the list. All fields except the stock code can be changed. Unsaved changes prompt a confirmation before leaving.
Changes are tracked in the Audit Log tab — who changed what and when.

7
Admin
Deactivate equipment
Click the power icon (⏻) on the list row. A confirmation modal appears. Deactivated equipment is hidden from active filters by default but is never deleted — it can still be found by switching the status filter to Inactive.
